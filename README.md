## Các mô đun cần
- Esp32c3 super mini
- đế ra chân cho esp ( bạn có thể tự hàn nếu có thể )
- các mô đun cảm ứng TTP223 ở đây dùng 4 cái
- màn oled 0.96 inch
- cảm biến DHT11
- các dây bus
- pin lipo
- màn eink 2.13 inch
- pin 3.7 v
- cảm biến ánh sáng TEMT6000
- còi buzzer
